# Contributing to DatasetVision

First off, thank you for considering contributing to DatasetVision! It's people like you that make DatasetVision such a great tool.

## How Can I Contribute?

### Reporting Bugs
- Use the [GitHub Issue Tracker](https://github.com/nibir-ai/datasetvision/issues).
- Describe the bug and include steps to reproduce.
- Include your OS and Python version.

### Suggesting Enhancements
- Open a new issue and describe the feature you'd like to see.
- Explain why it would be useful.

### Pull Requests
1. Fork the repository and create your branch from `main`.
2. Install development dependencies: `pip install -e '.[dev]'`.
3. If you've added code that should be tested, add tests.
4. Ensure the test suite passes: `pytest tests/`.
5. Make sure your code follows the existing style (we use Black).
6. Issue that pull request!

## Styleguide
- Use clear and descriptive commit messages.
- Add docstrings to new functions and classes.
- Use type hints wherever possible.
